This project started with a codebase that contained the environment setup, basic game actions, and some code for a reinforcement learning agent. The code gave the maze structure, state representation, and reward setup so the agent could work in the setting. I had to create the decision-making part using deep Q-learning.

I made the neural network design, put in the Deep Q-Network (DQN) method, used experience replay, and set values like learning rate, discount factor, and exploration rate. I also changed the training loop to make convergence better and changed the epsilon-greedy plan to balance exploration and taking action. I tested many things, like increasing training epochs, and improved the model until the agent could find the best way to the treasure all the time.
